HTML TAGS & HEIRARCHY

- by Heather Maloney

Coding HTML can be a tricky business. Below you will find a cheat sheet for basic HTML tags and heirarchy to help you along the way.

Basic Tags

Tag Name Open <> Close <> Syntax Example <> Description
HTML Start html /html Denotes beginning and ending of HTML code.
Head head /head Put invisible date here – metadata, website tab title, links to stylesheet, etc.
Body body /body Put visible webpage data between tags - everything from header to footer.
Header header /header Data that appears at the very top of the page.
Division div /div Section/frames that appear on a page - can adjust size and appearance individually. Can also give sections ID names.
Headings h[1-6] /h[1-6] Section headings. Sizes range from 1 (largest) to 6 (smallest).
Paragraph p /p Paragraph content.
Unordered List ul /ul Opens bullet point list.
Ordered List ol /ol Opens numbered list.
List Detail li /li Details each item in list.
Comments !--comment here--> Insert INVISIBLE comments about your coding.

Insertion Tags

Tag Name Open <> Close <> Syntax Example <> Description
Anchor Hyperlink a /a a href=”[insert hyperlink]”>Title of Hyperlink
Image img /img img src=”[insert image link]” Insert image. Good practice to add alt=”[description of image]” in opening tag for ADA purposes.
Video video /video Inserts ONLY video. Add controls style in tag open.
Audio audio /audo Inserts audio files. Add controls style in tag open.
Embed embed /embed Insert audio, video, images, etc. Add controls style in tag open for audio or video.

Formatting Tags

Tag Name Open <> Close <> Syntax Example <> Description
Attributes - Names/Values name="value" Located inside opening tag. Examples follow:
ID id="id-name] Give element an ID name. Good for navigation and styling.
Style style="color: green;" Add formatting to line items/sections instead of using CSS.
